How to get the Channel ID:
If your channel URL does not display the Channel ID, right-click on the channel's page and choose "View Page Source" (or press Ctrl+U). Then, use Ctrl+F to search for "channelId" or "data-channel-external-id". The Channel ID will typically start with UC.
How to get your YouTube Data API v3 API Key:
1. Go to the Google Developers Console.
2. Create a new project or select an existing project.
3. Navigate to the "Library" tab and search for "YouTube Data API v3".
4. Click on "Enable" to enable the API for your project.
5. Go to the "Credentials" tab and click on "Create Credentials".
6. Select "API key" from the dropdown menu. Your API key will be generated.
7. Copy the API key and paste it in the box above.
